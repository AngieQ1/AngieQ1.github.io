科普：双查询注入

当order by判断完字段数，但是union select 1,2,3,…不在页面显示返回位时。可以尝试双查询注入。

双查询注入就是嵌套子查询，简单说就是select(select)的形式。

数据库查询要么返回正确的查询结果，要么返回报错信息。双注入查询其实是一种报错查询，将我们需要的信息当作数据库错误返回。

这里利用了Error Based分析过的floor(rand(0)*2)引起报错的原理。
